# My Dotfiles



## How to use

### Linux

```
~]$ cd
~]$ rm dotfiles
~]$ git clone https://github.com/kuritaka/dotfiles
~]$ cd dotfiles/linux
~]$ chmod +x dotfiles-deploy.sh
~]$ ./dotfiles-deploy.sh
```
