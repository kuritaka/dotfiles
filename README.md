# My Dotfiles



## How to use

### Linux

```
~]$ cd
~]$ test -e dotfiles && mv dotfiles dotfiles.`date -d '1day ago' +%Y%m%d`
~]$ git clone https://github.com/kuritaka/dotfiles
~]$ cd dotfiles/linux
~]$ chmod +x dotfiles-deploy.sh
~]$ ./dotfiles-deploy.sh
```
